About Me

Halo, nama saya Supriyandi Ridzkia, seorang mobile developer dengan lebih dari 4 tahun pengalaman di bidang pengembangan aplikasi mobile. Saya memiliki spesialisasi dalam pengembangan aplikasi Android menggunakan Java serta Framework Flutter untuk aplikasi lintas platform.

Saya juga memiliki pengalaman dalam pengembangan REST API menggunakan PHP Native maupun framework seperti Go-Lang, CodeIgniter dan Laravel. Selain itu, saya berpengalaman dalam menggunakan JavaScript dan framework ExpressJS untuk membangun API yang cepat dan efisien. Saya juga menguasai penggunaan database MsSQL, MySQL, dan Firebase untuk merancang dan mengelola struktur data yang efektif dan scalable.

Dalam setiap proyek, saya selalu mengutamakan pengalaman pengguna dan memastikan bahwa aplikasi tidak hanya menarik secara visual tetapi juga berfungsi optimal. Dan saya selalu berharap dapat terlibat dalam lebih banyak proyek yang berdampak besar bagi masyarakat.
